# Android to Android

This APK is a tool for demonstrating and simulating attacks between two Android devices. It is intended for educational or entertainment purposes and does not contain any real exploits or malicious code.

## Features

*   **USB Attack:** Simulates a USB-based attack on a connected Android device, pretending to use ADB to gain access and execute commands.
*   **Network Attack:** Scans a target IP address for common open ports on Android devices.
*   **Reverse Shell:** Generates a reverse shell payload to be used on the target device.
*   **Data Exfiltration:** Simulates the exfiltration of sensitive data from the target device by listing common paths to sensitive files.
